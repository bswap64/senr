SENR
---------

This addon is a modified `lua/menu/errors.lua` file.                             
The block that shows script error notifications has been commented out, so errors will no longer appear.


## Addon showcase
![as](https://s12.gifyu.com/images/bkkit.gif)

## Installation
First, download the archive from this link: https://github.com/vulkansdk/senr/releases/download/senr/senr.7z  
Then, open the archive and move the `lua` folder into your `GarrysMod/garrysmod` directory.
